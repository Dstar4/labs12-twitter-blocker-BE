## **Add a new user **

_url_: `/users/`

_http method_: **[POST]**

#### Response

##### 201 (created)

###### Example input

```
{
	"twitter_id":"34519916395",
	"screen_name":"eblees4",
  "is_paying": true, //optional
  "token": "[token goes here]",
  "upvote": 5, //optional
  "downvote": 2, //optional
  "email": "email@email.com", //optional
  "admin": true, //optional
  "deactivated": true //optional
}
```

###### Example response

```
{
    "message": "User has been added"
}
```
